## ❗주요 기능

#### 회원가입
* 약관동의
* 아이디 중복확인
* PW 암호화 (bcrypt 사용)
* PW , PW 확인 값 일치 체크
* 휴대전화 인증
* DAUM 주소api 사용
* 필수값 모두 확인 후 회원가입 진행
